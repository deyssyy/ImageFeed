# ImageFeed 📱

Проект мобильного приложения с бесконечной лентой картинок, интегрированный с популярным сервисом Unsplash.

---

### 📝 Описание проекта
Приложение позволяет пользователям просматривать фотографии высокого качества в виде бесконечной ленты, искать публикации, ставить лайки и просматривать детальную информацию о каждом снимке. Работа с контентом происходит в реальном времени через официальное API Unsplash.

### ✨ Ключевые возможности (Features)
- **Авторизация**: Безопасный вход в аккаунт через протокол OAuth 2.0.
- **Бесконечная лента**: Плавная подгрузка изображений по мере скролла (Pagination).
- **Детальный просмотр**: Открытие фотографии во весь экран с возможностью зума и шаринга.
- **Профиль пользователя**: Отображение информации о текущем пользователе (аватар, имя, никнейм).
- **Интерактив**: Возможность ставить и убирать лайки с мгновенным обновлением состояния.

---

### 🛠️ Стек технологий и архитектура

- **Язык и UI:** `Swift`, `UIKit` (верстка кодом)
- **Архитектура:** `MVP` (чистое разделение логики и интерфейса)
- **Сеть и Данные:** `URLSession`, `REST API`, `JSON` (парсинг данных через Codable)
- **Безопасность:** `OAuth 2.0`, `Keychain` (надежное хранение токенов авторизации)
- **Многопоточность:** `GCD` (асинхронная загрузка данных без фризов UI)
- **Зависимости:** `Kingfisher` (оптимизированное кэширование и загрузка изображений)
- **Тестирование:** `XCTest` (Unit-тесты для сетевого слоя и UI-тесты)

### 🚀 Запуск и установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/deyssyy/ImageFeed
   ```
2. Откройте проект в Xcode:
   ```bash
   open ImageFeed.xcodeproj
   ```
3. Для полноценной работы сетевого слоя добавьте ваши ключи доступа Unsplash API в конфигурационный файл приложения.
Необходимо создать аккаунт на сайте [unsplash](https://unsplash.com) и зарегистрировать там свое приложение Image Feed. Затем в файле AuthConfiguration указать свои access_ket и secret_key, полученные после регистрации приложения
---

### 📐 Реализованные инженерные подходы
- Соблюдение принципов `SOLID`, `KISS`, `DRY` для чистоты и масштабируемости кода.
- Обработка ошибок сетевых запросов и состояний отсутствия интернета.
- Оптимизация переиспользования ячеек `UITableView` для плавной анимации ленты.
